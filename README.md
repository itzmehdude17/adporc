# ADPORC – Asia Digital Physiotherapy & Orthopedic Rehabilitation Center
A responsive and user focused website developed for ADPORC, one of Dhaka’s leading physiotherapy and orthopedic rehabilitation centers. The platform is designed to improve digital presence, patient awareness, and appointment accessibility.

Built using HTML, CSS, and JavaScript, the website delivers a fast, clean, and mobile friendly experience. It showcases core rehabilitation services including pain management, paralysis rehabilitation, and sports injury care, while clearly presenting physiotherapist profiles, clinical expertise, and treatment philosophy.

Key features include structured service sections, educational blog content for public health awareness, and a streamlined appointment booking system to reduce friction between patients and clinicians. The layout emphasizes clarity, accessibility, and trust, aligning with healthcare communication standards.

This project demonstrates practical frontend development for real world medical use cases, with a focus on usability, responsiveness, and professional presentation in a healthcare setting.
